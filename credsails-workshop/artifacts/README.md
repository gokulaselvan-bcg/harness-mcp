# Artifacts

This directory holds design and planning artifacts for the project.

- Use `artifacts/architecture/` for architectural decisions and ADRs.
- Use `artifacts/blueprints/` for project-level technical blueprints (HLD) and decomposed ticket sets.
- Use `artifacts/tickets/` for ticket-level designs and implementation plans.
- See `docs/src/content/docs/artifacts-guide.md` for the full workflow.
