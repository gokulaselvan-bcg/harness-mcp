# /blueprint - Technical Blueprint (HLD)

Create a project-level technical blueprint from initiative context and requirements.

## Usage

```bash
/blueprint [optional initiative context]
```

## Forwarding

Invoke the `blueprint` skill and follow `.claude/skills/blueprint/SKILL.md`.

## Inputs

- Initiative context and references (PRD/docs/diagrams)

## Output

- `artifacts/blueprints/{id}--{name}/blueprint.md`
