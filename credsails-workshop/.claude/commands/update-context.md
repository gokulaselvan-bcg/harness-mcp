# /update-context - Context maintenance

Update and reorganize `.claude/context/` for selective loading and navigation.

## Usage

```bash
/update-context [path|topic|area]
```

## Forwarding

Invoke the `update-context` skill and follow `.claude/skills/update-context/SKILL.md`.

## Inputs

- Optional path or topic scope

## Outputs

- Updated context indexes and files under `.claude/context/`
