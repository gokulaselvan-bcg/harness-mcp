# /prd - Product Requirements Document

Create a product requirements document for an initiative or feature.

## Usage

```bash
/prd [optional feature name]
```

## Forwarding

Invoke the `prd` skill and follow `.claude/skills/prd/SKILL.md`.

## Inputs

- Feature context, goals, scope, and references

## Output

- `artifacts/requirements/{feature-name}-prd.md`
