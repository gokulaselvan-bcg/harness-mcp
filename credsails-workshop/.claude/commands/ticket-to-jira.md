# /ticket-to-jira - Export ticket set to Jira

Export blueprint-generated epics and tickets to Jira via Atlassian MCP.

## Usage

```bash
/ticket-to-jira [optional tickets-file]
```

## Forwarding

Invoke the `ticket-to-jira` skill and follow `.claude/skills/ticket-to-jira/SKILL.md`.

## Inputs

- Tickets bundle from `artifacts/blueprints/*/tickets/`

## Output

- Jira epics and tasks created/linked from the selected ticket set
